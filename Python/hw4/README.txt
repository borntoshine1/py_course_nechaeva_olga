Tuple неизменяемые, но могут содержать изменяемые объекты, используют круглые скобки, 
list можно изменять, используют квадратные скобки.
Tuple быстрее работают, их не надо копировать как списки.
Если нужен массив неизменяемых данных выберу Tuple[], если данные в списке нужно изменять выберу list().

 lambda это функция в функции, которые не связаны с именем (анонимные). 
 lambda функции не включает оператор “return” всегда содержит выражения, создает очень компактный и по прежнему читаемый код.
 Используется внутри любых нормальных функций python.
 lambda может принимать много аргументов, но возвращает только одно выражение.
